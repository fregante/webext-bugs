# webext-bugs/reload-on-cmd-r

Safari doesn't support CMD-R (or Ctrl-R) to reload extension pages like the options page. Import this file to add that shortcut.

```js
// In options.js or any other extension page
import 'webext-bugs/reload-on-cmd-r';
```

## Compatibility

- Safari (other browsers are automatically ignored)

## Permissions

- No special permissions

## Context

- Extension pages
- Options page
